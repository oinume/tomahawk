tomahawk とは?
--------------
tomahawk は以下の3つを可能にするソフトウェアである。

* コマンドを複数のリモートホストに対して実行する
* ローカルのファイルを複数のリモートホストにコピーする
* 複数のリモートホストにあるファイルをローカルにコピーする
